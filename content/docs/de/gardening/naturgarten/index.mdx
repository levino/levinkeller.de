# Naturgarten auf 2.500 m²

Willkommen in der laufenden Dokumentation meines Gartenprojekts. Aus einer
vorwiegend aus Rasen bestehenden Fläche entsteht Schritt für Schritt ein
moderner Naturgarten mit Staudenbereichen, Wildsträuchern und Gemüsegarten.

## Struktur dieser Dokumentation

- [Ausgangslage und Leitidee](/de/docs/gardening/naturgarten/ausgangslage)
- [Bisherige Entwicklung](/de/docs/gardening/naturgarten/entwicklung)
- [Staudenbeete im Bestand anlegen](/de/docs/gardening/naturgarten/staudenbeete-im-bestand)
- [Roadmap und nächste Kapitel](/de/docs/gardening/naturgarten/roadmap)

Diese Struktur ist bewusst modular: Zu jedem Bereich kann ich künftig neue
Unterkapitel ergänzen, ohne dass ein einzelner Text zu lang oder unübersichtlich
wird.
